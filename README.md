# Projecte Final MERN - Receptes App

Aplicació web per gestionar receptes utilitzant MongoDB, Express, React i Node.js.

---

## Com executar el projecte

### 1. Clonar el repositori

```bash
git clone git@github.com:souhailaezzarfi/ProjecteFinalMERN.git
cd ProjecteFinalMERN
```

---

## Backend (server)

### 2. Entrar a la carpeta del servidor

```bash
cd server
```

### 3. Crear arxiu d'entorn

```bash
cp .env.example .env
```

### 4. Instal·lar dependències

```bash
npm install
```

### 5. Executar el servidor

```bash
npm run dev
```

Backend executant-se a:

```
http://localhost:3005
```

---

## Frontend (client)

### 6. Obrir una nova terminal i anar al client

```bash
cd client
```

### 7. Instal·lar dependències

```bash
npm install
```

### 8. Executar el frontend

```bash
npm run dev
```

El frontend s'executarà a:

```
http://localhost:5173
```

---

## Notes importants

- Cal crear el fitxer `.env` a partir de `.env.example`
- El backend utilitza el port `3005`

---

## Tecnologies

- MongoDB
- Express
- React (Vite)
- Node.js
